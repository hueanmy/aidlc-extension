A VS Code extension that turns AI agents into a real SDLC team — 9 phases, clear ownership, auto-wired to Claude Code.

Epic Tree View Pipeline Dashboard MCP auto-config
Pipeline · Quy trình docs/sdlc/epics/EPIC-1000
01 Plan Lập kế hoạch
AgentProduct Owner ArtifactEpic + PRD

Turns business goals into an epic and PRD with clear scope and acceptance criteria. Chuyển mục tiêu business thành epic và PRD, phạm vi rõ, tiêu chí nghiệm thu.

02 Design Thiết kế kỹ thuật
AgentTech Lead ArtifactTech Design

Picks architecture, data flow, and trade-offs before a single line of code is written. Chọn kiến trúc, luồng dữ liệu, đánh đổi — trước khi viết dòng code nào.

03 Test Plan Kế hoạch kiểm thử
AgentQA Engineer ArtifactTest Plan

Defines test cases, coverage goals, and risk areas up-front — not after the bug ships. Định nghĩa test case, độ phủ, vùng rủi ro — chứ không đợi bug ra prod mới viết.

04 Implement Triển khai
AgentDeveloper ArtifactFeature branch

Writes code against the tech design and test plan; every commit traces back to the epic. Viết code bám tech design và test plan; mỗi commit truy ngược về epic.

05 Review Đánh giá code
AgentTech Lead ArtifactApproval

Checks design fit, edge cases, and code quality before merge — not just lint. Kiểm tra khớp thiết kế, edge case, chất lượng code — không chỉ chạy lint.

06 UAT Kiểm thử nghiệm thu
AgentQA Engineer ArtifactUAT Script

Runs the acceptance script with the PRD in hand; sign-off only when criteria are met. Chạy script nghiệm thu với PRD; chỉ sign-off khi đạt đủ tiêu chí.

07 Release Phát hành
AgentRelease Manager ArtifactGit tag

Cuts the tag, drafts release notes, and owns the rollback playbook if things break. Cắt tag, soạn release note, giữ playbook rollback phòng khi có sự cố.

08 Monitor Giám sát
AgentSRE ArtifactHealth report

Watches metrics and alerts after ship; feeds regressions back to Plan. Theo dõi metric và alert sau release; phản hồi ngược về giai đoạn Plan.

09 Doc Sync Đồng bộ tài liệu
AgentArchivist ArtifactReverse-sync doc

Reverse-syncs what actually shipped back into the docs — truth stays with the code. Đồng bộ ngược những gì thực sự ship về docs — sự thật luôn bám theo code.
